# UI/UX Pro Max Skill

专业的 UI/UX 设计智能技能,为 AI 编码助手提供全面的设计知识库。

## 功能特性

- **67+ UI 风格**: 从极简主义到粗野主义、玻璃拟态等
- **96+ 行业配色方案**: 针对 SaaS、电商、医疗、金融等行业
- **57+ 字体搭配**: 经过验证的专业字体组合
- **99+ UX 最佳实践**: 涵盖导航、表单、按钮、内容层次等
- **25+ 图表类型**: 适合数据可视化的图表选择指南
- **13+ 技术栈支持**: HTML/Tailwind、React、Vue、Flutter 等

## 使用方法

### 自动触发

当你提出任何 UI/UX 相关的设计需求时,此 skill 会自动激活,例如:

- "设计一个现代化的登录页面"
- "帮我选择合适的配色方案"
- "优化这个页面的用户体验"
- "创建一个数据仪表板"
- "改进移动端的响应式布局"

### 手动调用

你也可以明确指定使用此 skill:

```
使用 ui-ux-pro-max skill 设计一个宠物用品电商的首页
```

## 工作流程

当 skill 激活时,会按照以下步骤进行设计:

1. **识别产品类型和行业** - 确定目标用户和业务场景
2. **选择 UI 风格** - 从 67+ 专业风格中选择最合适的
3. **挑选配色方案** - 基于行业特点选择 96+ 配色之一
4. **确定字体搭配** - 从 57+ 字体组合中选择
5. **应用 UX 准则** - 遵循 99+ 用户体验最佳实践
6. **组件模式选择** - 使用经过验证的组件设计模式
7. **技术栈实现** - 根据项目技术栈生成代码
8. **质量检查** - 确保符合可访问性、响应式等标准

## 避免的设计陷阱

- ❌ 过度使用紫色-粉色渐变(AI 审美)
- ❌ 所有元素都使用大圆角
- ❌ 颜色对比度不足
- ❌ 导航结构混乱
- ❌ 表单体验差
- ❌ 缺少加载状态和错误提示
- ❌ 移动端适配不良

## 注意事项

1. **简洁至上** - 设计应该服务于功能,不要过度装饰
2. **一致性** - 保持整个应用的视觉语言统一
3. **可访问性** - 确保所有用户都能使用你的产品
4. **性能优先** - 美观不应牺牲加载速度和交互流畅度
5. **用户中心** - 始终以用户需求为设计出发点
